SYSTEM PREREQUISITES
To use this SDK you must already have a functional Apache Server installations configured to with PHP. And a project directory defined.

INSTALLATION NOTES
1. Copy the folder named class into a convenient location in your project folder

2. Include /path/to/project/directory/class/BSGateway.php in the script from which you will be calling the class file

3. Refer to example.php for sample calls to the bsgateway class.
